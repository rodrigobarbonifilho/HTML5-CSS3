Semântica

De acordo com o dicionário: num sistema linguístico, o componente do sentido das palavras e da interpretação das sentenças e dos enunciados

Principais formatações

Negrito / Destaque

Nesta frase temos um termo em negrito usando a tag <B> (não semântica)

Nesta frase temos um termo em negrito usando a tag <STRONG> (semântica)

Itálico / Enfâse

Nesta frase temos um termo em Enfâse usando a tag <I> (não semântico)

Nesta frase temos um termo em Enfâse usando a tag <EM> (semântico)

Textos marcados

Podemos criar um texto marcado usando a tag <MARK>

Também a tag <MARK> estilizada com CSS: texto marcado

Texto Grande e Pequeno

estamos criando umt texto grande e um texto pequeno. Mas o <BIG> é uma tag que não faz sentido, pois, dá para se fazer isso com CSS, por isso está obsoleta.

Texto deletado

Podemos marca um texto com excluído para indicar que ele deve ser lido, mas não considerado. Usando a tag: <DEL>

Texto Inserido

Podemos marcar um texto como inseido para dar uma ênfase e indicar que ele foi adicionado depois. Usando a tag <INS>

Podemos fazer isso com a Tag <U> (não semântica)

Texto sobrescrito

Para inserir coisas do tipo x20+3

Texto subscrito

Para inserir coisas do tipo: H20 (formula da água)
